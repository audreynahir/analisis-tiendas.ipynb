# analisis-tiendas.ipynb
# 📊 Análisis de rendimiento de tiendas

## 📌 Descripción del proyecto

Este proyecto presenta un análisis de datos realizado en **Google Colab** utilizando Python y la librería **Pandas**.

El objetivo del proyecto es analizar el rendimiento de cuatro tiendas a partir de diferentes indicadores de negocio para determinar cuál sería la mejor opción para que el Sr. Juan venda.

Los datos analizados contienen información sobre ventas, productos, categorías, precios, costos de envío, calificaciones de clientes y ubicación de las compras.

---

## 🎯 Objetivo

Evaluar el desempeño de cuatro tiendas utilizando análisis de datos para identificar:

- Facturación total
- Categorías más vendidas
- Satisfacción de los clientes
- Productos más y menos vendidos
- Costos promedio de envío

Con base en estos indicadores se busca recomendar qué tienda debería vender el Sr. Juan.

---

## 📂 Datos utilizados

El conjunto de datos contiene las siguientes variables:

- Producto  
- Categoría del producto  
- Precio  
- Costo de envío  
- Fecha de compra  
- Vendedor  
- Lugar de compra  
- Calificación  
- Método de pago  
- Cantidad de cuotas  
- Latitud  
- Longitud  

Los datos corresponden a ventas realizadas en cuatro tiendas diferentes.

---

## 🔎 Análisis realizados

### 1. Análisis de facturación
Se calculó el total de ingresos generados por cada tienda para identificar cuál presenta mayor volumen de ventas.

### 2. Ventas por categoría
Se analizaron las categorías de productos con mayor y menor volumen de ventas.

### 3. Calificación promedio de la tienda
Se evaluó la satisfacción de los clientes mediante el promedio de calificaciones por tienda.

### 4. Productos más y menos vendidos
Se identificaron los productos con mayor y menor número de ventas.

### 5. Costo promedio de envío
Se compararon los costos de envío entre las diferentes tiendas.

---

## 🛠 Herramientas utilizadas

- Python

- GitHub

---

## 📈 Resultados principales

Los resultados del análisis muestran que:

- **Tienda 1** presenta la mayor facturación total.
- **Tienda 3** tiene la mejor calificación promedio por parte de los clientes.
- Las categorías con mayores ventas son **Electrónicos** y **Electrodomésticos**.
- **Tienda 4** presenta el menor costo promedio de envío.

---

## ✅ Conclusión

Tras analizar todos los indicadores, se concluye que **la Tienda 3 representa una opción equilibrada**, ya que combina una buena facturación con la mejor calificación de clientes y costos de envío competitivos.

Por esta razón, se recomienda que el Sr. Juan considere vender la **Tienda 3**, ya que presenta un balance favorable entre ingresos, satisfacción del cliente y eficiencia operativa.
